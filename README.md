# Amazon-Sales-Data-Analysis

# Objective
Analyze Amazon sales data to understand trends, customer behavior, and product performance.

# Tools Used
- Excel / CSV
- Power BI

# Key Insights
- Sales trends were stable over time  
- Balanced category performance  
- Wide customer distribution across regions 

# Conclusion
I analyzed an Amazon sales dataset using Power BI by cleaning data, creating calculated fields, and building an interactive dashboard. It showed stable sales trends, balanced category performance, and a wide customer base, helping improve inventory planning and business decisions.
The analysis helps in better inventory planning and supports data-driven business decisions.
